Features

Sizes

Bootstrap toggle is available in different sizes.

Custom Sizes

Bootstrap toggle can handle custom sizes by data-width and data-height options.

Tristate Toggle

Simply add tristate to enable a three state toggle.

Colors

Bootstrap Toggle implements all standard bootstrap 5 button colors.

Outline Colors

Bootstrap Toggle implements all standard bootstrap 5 button outline colors.

Custom Style

Style the buttons to fit an existing UX.

Custom Text

The text can be changed easily with data attributes data-onlabel and data-offlabel.

CHANGE v5.0.0 Using data-on and data-off data attributes is deprecated and will throw a console warning. Use data-onlabel and data-offlabel data attributes instead.

HTML, Icons, Images

You can easily add icons or images since html is supported for on/off text.

Multiple Lines of Text

Toggles with multiple lines will adjust its heights.

Custom title

NEW v5.0.0

Simply add data-ontitle and data-offtitle attributes to set a customs titles to each par od the toggle. To set the same title in both parts just use HTML standard title attribute.

Custom tabindex

Simply add tabindex="x" to set tabindex to x. If not tabindex is provided it will be set to 0. Toggle can be switched pressing SPACEBAR while it's focused.

Support for labels

NEW v5.0.0

The label for the original checkbox is redirected to the bootstrap toggle element. Clicking the label will switch and focus the toggle.

Animation Speed

Transition speed can be easily controlled with css transition property on .toggle-group. You can also turn animation off completely.

State and Status

Toggle atate and status is simply set adding vanilla html attributes to the checkbox:

  • Simply add checked to convert checkboxes into on toggles.
  • Simply add disabled to convert checkboxes into disabled toggles. Checkboxes will not be presents in the payload of a submited form.
  • Simply add readonly to convert checkboxes into disabled toggles but presents in the payload of a submited form.
Payload:
<input name="enabled" type="checkbox" data-toggle="toggle" checked>
<input name="disabled" type="checkbox" data-toggle="toggle" checked disabled>
<input name="readonly" type="checkbox" data-toggle="toggle" checked readonly>

Form submit values

Simply add data-onvalue="x" data-offvalue="y" to set on state value to x and off state value to y. If original checkbox have a value settes, data-onvalue will be ignored preserving the original value.

Payload:
<input type="checkbox" data-toggle="toggle" checked value="VALUE" data-onvalue="ON" data-offvalue="OFF" name="valueToggle">
<input type="checkbox" data-toggle="toggle" checked data-onvalue="ON" data-offvalue="OFF" name="dataONToggle">
<input type="checkbox" data-toggle="toggle" checked name="simplyToggle">

Stacked Checkboxes

Simply add data-toggle="toggle" to convert checkboxes into toggles inside a Bootstrap form-check.

Inline Checkboxes

Simply add data-toggle="toggle" to a convert checkboxes into toggles inside a Bootstrap form-check form-check-inline.

Input-Group Checkboxes

Simply add data-toggle="toggle" to convert checkboxes into toggles inside a Bootstrap input-group.
